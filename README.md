# Guard Stats

Follow up and manage the status of your [AdGuard Home](https://github.com/AdguardTeam/AdGuardHome) instances with this native iOS and macOS app.

Monitor your AdGuard Home instances right from your iPhone, iPad, or Mac menu bar. Guard Stats is a fork of [Pi Stats](https://github.com/Bunn/PiHoleStats) adapted for AdGuard Home, providing a beautiful, modern interface to keep track of your network-wide ad blocking statistics.

## Features

- Real-time monitoring of AdGuard Home statistics
- Quick enable/disable protection controls
- Native iOS app with widgets support
- macOS menu bar app for quick access
- Secure credential storage in Keychain
- Beautiful charts and metrics
- Support for multiple AdGuard Home instances

## Pi Monitor
For optional monitoring of your Raspberry Pi, you can install [Pi Monitor](https://github.com/Bunn/pi_monitor). Once installed, enable it on Guard Stats for enhanced performance insights and real-time statistics.


## Authentication

To use the enable/disable protection button, add your AdGuard Home password in the Settings screen.

Use the same password you use to log into the AdGuard Home web interface (default port: 3000).

If your AdGuard Home username is not `admin`, enter credentials as `username:password` in the password field.


## Requirements

- **iOS**: iOS 26.0 or later (iPhone, iPad)
- **macOS**: macOS Sequoia (15.2) or later
